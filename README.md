# Heliotrax-Reference-Firmware

## Overview
This repository contains reference firmware for the Heliotrax hardware platform.

It is intended for:
- Hardware testing
- Proof of concept
- Customer guidance

## Important
This is NOT production-ready software.
Customers are expected to develop their own applications.

## Structure
- /transmitter – sends data
- /receiver – receives data

## Getting Started
1. Install Arduino IDE
2. Open transmitter code
3. Upload to device
